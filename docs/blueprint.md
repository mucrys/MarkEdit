# **App Name**: MarkEdit

## Core Features:

- Markdown Rendering: Render Markdown documents for easy viewing on iOS devices.
- Markdown Editing: Allow users to edit Markdown documents with a simplified iOS-friendly interface.
- Local Storage: Enable users to store Markdown documents locally on their devices.
- File Import/Export: Allow users to import and export Markdown files to other applications or cloud storage.
- Syntax Highlighting: Highlight the syntax inside Markdown for better readability and coding.
- AI Assistance for text Generation: Help users rephrase or rewrite some of their documents by applying generative AI tool .

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to give a professional yet modern feel.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: Soft Lavender (#D8B4FE) for subtle highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font for clean readability and modern appearance.
- Code font: 'Source Code Pro' for displaying markdown and code snippets.
- Use minimalist icons to maintain a clean and distraction-free interface.
- Subtle animations for transitions and feedback, avoiding distracting effects.
